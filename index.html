Android App to Play Music Stored in Firebase:

GitHub Link(This repository is still private. The app is getting its finishing touch.)

Technologies used: Android Studio, Java, Firebase.

In this project, first songs are uploaded to firebase storage. Second, the songs list with the song names, singer, album artist are added in firebase realtime database along with the links of the songs stored in the storage. Then the app has a recycler view to list the songs available in the realtime database. When pressed to any adapter of the recyclerview which is used to put the song names, a background thread AsyncTask loads the song and play using android mediaplayer API. I have also integrated YouTube API in my project to enable users to load, search and play YouTube videos from the app
